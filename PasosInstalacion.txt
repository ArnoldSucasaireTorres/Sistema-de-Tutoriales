1. Descargar python en su computadora la version será 3.10.0
2. Abrir la terminal en modo administrador y situarse en la ruta donde se encuentra el archivo get-pip.py
3. colocar el siguiente comando (python3 get-pip.py) o (py get-pip.py) para instalar pip
4. Crear el entorno virtual con el nombre (virtual) con el siguiente comando: py -m venv virtual, al mismo nivel del archivo get-pip.py
5. activar el entorno virtual para instalar django, colocar en el terminal lo siguiente: 'virtual\Scripts\activate.bat'
6. una vez este activado el entorno virtual, se le mostrará en el prompt del terminal al inicio el nombre de (virtual), ejecuten la siguiente instruccion en el terminal, 'pip install -r requirements.txt' , esto instalará todas las dependencias necesarias para el proyecto de CS
7. Para probar que todo funciona correctamente entrar al proyecto SistemaTutoriales y ejecutar el comando 'py manage.py runserver' e ingresar al navegador con la siguiente url : http://127.0.0.1:8000/prueba/

Nota: para desactivar el entorno virtual usar el comando 'deactivate' , cada vez que se vuelva a usar el proyecto ejecutar el paso 5
